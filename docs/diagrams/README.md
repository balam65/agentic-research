# Agentic Research Diagrams

We use **Mermaid** diagrams stored as `.mmd` files for architectural flowcharts.

This directory houses the structural models mapped out in the `agentic-research` implementation documentation.

## Registry

`registry.json` maintains the mapping from diagrams to target implementation markdown files. This ensures consistency across the platform if diagram embedding scripts are utilized.

## Available Diagrams

- `agentic-research-architecture.mmd`
- `assessment-discovery-handoff.mmd`
- `extraction-scaling-pipeline.mmd`
- `research-orchestration-flow.mmd`
